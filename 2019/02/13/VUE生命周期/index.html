template

通过官方文档我们可以知道template也会对生命周期产生影响。

  • 如果vue实例对象中有template参数，则将其作为模板编译成render函数
  • 如果没有template参数，则将外部html作为模板编译
  • 通过此可以看出template的优先级要比outer- html要高

beforeMount和mounted 钩子函数间的生命周期

可以看到此时是给vue实例对象添加$el成员，并且替换掉挂在的DOM元素。

通过图1我们可以看到beforeMount时给实例对象添加$el成员，但是message里的内容还没有加载，以虚拟DOM的形式存在的。在mounted时，完成挂载。

通过图2我们可以看出render参数在beforeMount时并没有实现，认识在mounted时替换了此前的outer HTML，template也是同样的效果

beforeUpdate钩子函数和updated钩子函数间的生命周期

当vue发现data中的数据发生了改变时，会触发对应组建的重新渲染。

我们在console中输入：

app.message = 'update'

效果如下：

beforeDestroy和destroyed钩子函数间的生命周期

beforeDestory在实例销毁前调用，此时实例仍可使用

destoryed在实例销毁后使用，调用后，VUE实例所有东西都会解绑，所有时间监听器会被移除，所有的子实例也会被销毁。

生命周期的应用

beforeCreate

  • loading效果

created

  • 异步数据请求
  • 初始化数据

###mounted

  • 挂载元素内dom节点的获取
  • 可以配合$.nextTick进行单一时间对数据更新后更新DOM

update

  • 当数据更新需要做统一业务处理的时候使用
