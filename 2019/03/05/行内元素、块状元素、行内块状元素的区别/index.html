行内元素、块状元素、行内块状元素的区别

发表于 2019-03-05 | 分类于 -HTML
字数统计 | 阅读时长

区别

行内元素

行内元素特点：

  • 设置宽高无效
  • margin仅对left,right有效，top，bottom无效
  • padding有效
  • 不会自动进行换行

行内元素：

  • span
  • a
  • br
  • b
  • i
  • strong
  • em
  • input
  • textarea
  • select

块内元素

特点：

  • 能够识别宽高
  • margin和padding都有效
  • 可以自动换行
  • 多个块状元素自写在一起，默认排列方式从上至下

元素：

  • div
  • p
  • nav
  • ul
  • ol
  • li
  • form
  • table

行内块状元素

特点：

  • 不自动换行
  • 能够识别宽高
  • 默认排列方式从左至右
  • margin和padding都有效
