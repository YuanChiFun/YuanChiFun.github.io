NexT主题的安装与配置

安装NexT

  • 我们进入到博客的目录下，就是有themes文件夹的这个目录下，右键Cmd Here
  • 输入$ git clone curl -s https://api.github.com/repos/iissnan/hexo-theme-next/releases/latest | grep tarball_url | cut -d '"' -f 4 | wget -i - -O- | tar -zx -C themes/next --strip-components=1
  • 下载完成后我们打开目录下的_config.yml我们把theme改为NexT
    theme: next

    后面我们将会打开很多不能直接打开的文件，建议使用sublime或者Vscode打开。使我们后面的操作能变得更便捷些。

  • 做完了上面的这些，我们的博客就已经使用了NexT主题，大家可尝试在本地服务器上运行试试

  • 在浏览器中输入localhost:4000就可以查看了
